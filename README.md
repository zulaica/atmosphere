# Atmosphere

## Prerequesites
* [Yarn](https://yarnpkg.com/)

## Local Build Instructions
* `$ yarn`
* `$ yarn build`

* Use the imperative mood (e.g. "Do the thing" not "Does the thing")
* Start the commit message with the appropriate emoji pair (followed by two
